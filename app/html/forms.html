Overview

There are many options available: click to color the content to suit your content size, or use the radius feature. It can be used as a label, so apply it to your project!

Underline Inputs

Underline, the most powerful feature, uses the .u class together, it just colors the bottom part of the content and can be used as a button! If you want to use it as a button, it can be used together with the form tag. It is recommended to write it as a block element, and input tag or div tag is recommended. Be sure to apply a tabindex if you're writing div tags, or a .focuse class.

Checkboxes,Radios

Checkboxes and radios must use the input tag, which requires a parent tag that depends on the input and recommends block elements like the div tag. No extra-color features of the variable table are provided. I use .mars-form-checkbox, .mars-form-radio I use Javascript! It's a feature that controls checks, so it depends on jquery so always keep jquery up to date! Also, svg is included and if you want to customize Control it with fontawesome Mars uses fontawesome by default, adapt it to your project to suit your personality!
