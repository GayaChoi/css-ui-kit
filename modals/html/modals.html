Modals

modals are used with buttons and apply mars-modals. The structure of modals provides closing buttons, titles, information about content, mars-button.is-cancel, and mars-button.is-submit. Events must be implemented directly in JavaScript. mars does not provide. modals provides a .delete class that closes a modal window, which must be controlled with a button tag or block element.

We recommend that you use the parent tag of the block element for the element that will contain the title, see the modal-title-area class in the sample code. The title of modals uses is-title and can be used with color, see here for color. The content can be implemented after the title, which can be changed according to your project. The sample code is implemented after the title.

Next, it's the cancel and submit buttons, which must be controlled with the mars-button. Customization is not possible, but you can put your own ui or buttons with functions provided by mars-button. However, mars-modals has a width of 30rem, so please adjust the length or customize it according to your project.

Overview

modals keeps a default 30rem, delete comes in a variety of sizes by default. It is different from general mars-button size.Also, cancel and submit can be used as mars-button-icon. If you want to use the functions provided by mars, fontawesome is recommended.
