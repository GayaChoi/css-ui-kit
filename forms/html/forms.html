Inputs

The form in mars uses a class called mars-form Since mars is basically a block element, We recommend that you use a block element like div.

Overview

The mars-form input function can specify round, the default round does not apply.Each round has the following class

  • is-nor-rounded
  • is-md-rounded
  • is-lg-rounded

3px, 0.5em, 2em in order, customizable as needed. Next you can assign a color to the border-see the color palette for details.

Underline Inputs

mars-form-underline is applied to the border-bottom of the inputs, which fires an event on click. If you use a tag other than the input tag, you must apply the tabindex attribute. mars can apply an is-focused
is-actived class to its input applying it to mars-form-underline will invalidate the click event.

Overview

Checkboxes,Radios

The checkbox in mars applies a mars-form-checkbox, so you must use one input tag and one block element. It must first be placed inside the parent block element and must always remain as a child of the parent block element.

If you don't want to, start from mars-form-checkbox and implement your own internals.

The is-checked class should be applied to the child element input tag to locate the check icon. radio is very similar tomars-checkbox , using mars-form-radio and icons for radio-check-icon . Otherwise it is the same asmars-form-checkbox Next, the check icon applies the checkbox-check-icon class, so make sure to use block elements. You must set the type to checked in the input tag to use the check function.
